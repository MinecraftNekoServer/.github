## NPC创建和配置

### 基本创建命令
创建npc类型为玩家模型
```
/npc create 起床战争 PLAYER
```

### 基本配置命令
看向玩家
```
/npc lookatme 起床战争
```

设置居中
```
/NPC center 天坑乱斗
```

设置皮肤
```
/NPC skin 起床战争 dynamic DREAM
```

### 全息显示设置
添加内容（头顶显示）
```
/npc holo add 起床战争
```
可加颜色和多行，例如
```
/npc holo add 起床战争 &e&l起床战争
```

### 交互操作设置
添加指令操作。服务器传送
```
/npc action add 起床战争 LEFT COMMAND server Bwlobby
```

### 保存设置
保存ncp设置 必须！
```
/npc storage save
```

## 权限管理

### 管理员权限等级
- **初级管理员**: 负责日常维护和玩家管理
- **高级管理员**: 拥有服务器配置和插件管理权限
- **超级管理员**: 拥有所有权限，包括服务器核心设置

### 称号权限设置
```
prefix.0.&e&l【称号】&f
```

### 权限分配原则
1. 最小权限原则：只分配完成工作所需的最低权限
2. 职责分离：不同职责的管理员拥有不同的权限集合
3. 定期审查：定期检查和调整管理员权限

## 监控与性能

### 性能监控命令
- 查看TPS：`/tps`
- 查看内存使用：`/mem`
- 查看在线玩家：`/list`
- 查看延迟：`/ping`

### 日志监控
- 主日志文件：`logs/latest.log`
- 错误日志：`logs/error.log`
- 定期检查日志中的警告和错误信息

### 性能优化建议
1. 定期清理不必要的实体
2. 限制区块加载范围
3. 优化插件配置
4. 监控资源使用情况

## 故障排除

### 常见问题及解决方案

#### 服务器无法启动
1. 检查日志文件中的错误信息
2. 确认端口未被占用
3. 检查配置文件是否有语法错误

#### 玩家无法连接
1. 检查服务器是否在运行
2. 确认防火墙设置
3. 检查网络连接状态

#### NPC显示异常
1. 重新加载NPC插件：`/npc reload`
2. 检查NPC配置文件
3. 确认皮肤文件是否存在

#### 插件报错
1. 检查插件版本兼容性
2. 查看插件配置文件
3. 重新安装插件

### 紧急处理流程
1. 立即备份当前状态
2. 记录错误信息和时间
3. 尝试重启服务器
4. 联系技术支持（如需要）
